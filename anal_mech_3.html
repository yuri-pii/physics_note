Ⅰ. 解析力学で使う数学公式（最小セット）

1. 多変数関数の一次テイラー展開

\[ f(x+\varepsilon a,\,y+\varepsilon b) = f(x,y) + \varepsilon\left( \frac{\partial f}{\partial x}a + \frac{\partial f}{\partial y}b \right) +O(\varepsilon^2) \]

使う場面
    • 作用の変分
    • 「一次の変化だけを見る」ため

覚え方メモ

変分＝一次テイラー展開

これだけで9割の変分計算が始まる。

2. 変分の定義（方向微分）

\[ \delta S = \left.\frac{d}{d\varepsilon}S[q+\varepsilon\eta]\right|_{\varepsilon=0} \]

使う場面
    • 「δとは何か」を曖昧にしないため

覚え方メモ

δは「関数空間での微分」
時間微分とは別物。

3. 微分の線形性

\[ \delta \dot q = \frac{d}{dt}(\delta q) \]

使う場面
    • \dot q を含むラグランジアンの変分

覚え方メモ

δと d/dt は交換できる
（十分な滑らかさが前提）

4. 部分積分

\[ \int_{t_1}^{t_2} A\,\dot B\,dt = \left[A B\right]_{t_1}^{t_2} - \int_{t_1}^{t_2} \dot A\,B\,dt \]

使う場面
    • 変分計算の核心
    • \dot\eta を消す

覚え方メモ

E–L 方程式は
部分積分1回で生まれる

5. 変分法の基本補題

\[ \int_{t_1}^{t_2} f(t)\,\eta(t)\,dt = 0 \quad(\forall\,\eta,\ \eta(t_1)=\eta(t_2)=0) \Rightarrow f(t)=0 \]

使う場面
    • 積分条件 → 局所方程式への変換

覚え方メモ

「任意」なら「中身ゼロ」

6. ルジャンドル変換（エネルギー）

\[ H(q,p,t)=\sum_i \dot q_i p_i - L \]

使う場面
    • エネルギー定義
    • ハミルトニアンへの移行

覚え方メモ

速度 → 運動量への視点変更

Ⅱ. 解析力学の物理的公理・原理（意味付き）

1. 作用の停留原理（最重要）

\[ \delta S = 0, \qquad S[q]=\int_{t_1}^{t_2} L(q,\dot q,t)\,dt \]

物理的意味
    • 自然は「経路全体」を一度に比較する
    • 一次の違いが消える経路が実現する

覚え方メモ

自然は最適化しない
区別できない経路が残る

2. ラグランジアンの定義

\[ L = T - V \]

物理量の説明
    • T：運動エネルギー（速度の情報）
    • V：位置エネルギー（配置の情報）

単位
    • L：エネルギー [J]

覚え方メモ

L は「重要性の宣言」
何を保存し、何を無視するか。

3. オイラー＝ラグランジュ方程式

\[ \frac{d}{dt}\left(\frac{\partial L}{\partial \dot q_i}\right) - \frac{\partial L}{\partial q_i} =0 \]

物理的意味
    • 作用の停留条件の局所表現
    • 力学の運動方程式

覚え方メモ

E–L は原理ではない
公理の翻訳結果

4. 共役運動量の定義

\[ p_i := \frac{\partial L}{\partial \dot q_i} \]

物理量の説明
    • 座標 q_i に対応する運動量
    • 保存則の主役

単位
    • 通常の運動量：[kg·m/s]

覚え方メモ

「速度の重要度」を測る量

5. 循環座標の原理

\[ \frac{\partial L}{\partial q_i}=0 \Rightarrow \frac{dp_i}{dt}=0 \]

物理的意味
    • その方向に物理的区別がない
    • → 保存量が生まれる

覚え方メモ

対称性＝依存しない変数

6. Noetherの原理（超要約）

\[ \text{連続対称性 ↔ 保存量} \]

具体例

対称性 保存量
時間並進 エネルギー
空間並進 運動量
回転 角運動量

覚え方メモ

対称性は記憶を残す

7. エネルギー保存

\[ E=\sum_i \dot q_i p_i - L, \qquad \frac{\partial L}{\partial t}=0 \Rightarrow \frac{dE}{dt}=0 \]

物理量の説明
    • 時間方向の Noether 量

単位
    • エネルギー [J]

覚え方メモ

時間に依らない → 時間方向の保存

8. 有効ポテンシャル

\[ V_{\mathrm{eff}}(r)=V(r)+\frac{l^2}{2mr^2} \]

物理的意味
    • 消した自由度の「記憶」
    • 制約を守るためのエネルギー代

覚え方メモ

自由度削減の痕跡
